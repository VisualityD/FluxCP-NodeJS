Welcome! This is the latest version of FluxCP for the Ratnena emulator.This version is written in NodeJS using React.
At the moment, the site is not fully migrated and there are some bugs, but you can already familiarize yourself with this project.

1. You need to add the Rathena SQL database
2. Install the latest version of NodeJS
3. Launch CMD
4. Enter the command `npm install`
5. To start the project, enter:
Enter the command `npm run dev`
6. Go to the website: http://localhost:5173/
